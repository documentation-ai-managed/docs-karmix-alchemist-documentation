## Release History

<Update label="2024-10-15" description="v2.1.0" tags={["feature", "improvement"]}>

- Introduced document templates for common content types
- Enhanced search with fuzzy matching and filters

- Better mobile responsiveness across all devices
- Enhanced API rate limiting and error handling
