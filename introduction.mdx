## Key Features

Karmix Alchemist stands out with its intuitive interface and robust feature set. Our platform supports multiple documentation formats, including Markdown and rich text, allowing you to create content that matches your style.

<Tabs>
  <Tab title="Markdown Support" icon="file-text">
    Write documentation using familiar Markdown syntax with live preview capabilities.
    
    <Steps>
      <Step title="Create Document" icon="plus">
        Click the "New Document" button in your dashboard.
      </Step>
      <Step title="Write Content" icon="edit">
        Use Markdown syntax to format your content.
        
        <CodeGroup tabs="Basic,Advanced">
          ````markdown
          # My Documentation
          
          This is a simple paragraph.
          
          ## Subsection
          
          - List item 1
          - List item 2
          ````
          ````markdown
          # Advanced Features
          
          ```javascript
          console.log('Code blocks supported');
          ```
          
          > Blockquotes work too
          
          | Feature | Status |
          |---------|--------|
          | Tables | ✅ |
          | Code | ✅ |
          ````
        </CodeGroup>
      </Step>
      <Step title="Publish" icon="send">
        Save and publish your document for team access.
      </Step>
    </Steps>
  </Tab>
  <Tab title="Version Control" icon="git-branch">
    Track changes and maintain version history for all your documentation.
    
    <ExpandableGroup>
      <Expandable title="What is version control?" default-open="false">
        Version control allows you to track changes over time, collaborate with team members, and revert to previous versions if needed.
      </Expandable>
      <Expandable title="How does it work in Karmix?" default-open="false">
        Every save creates a new version automatically. You can view diffs, restore old versions, and see who made changes.
      </Expandable>
    </ExpandableGroup>
  </Tab>
  <Tab title="Search & Discovery" icon="search">
    Find information quickly with powerful search capabilities across all your docs.
    
    <Callout kind="tip">Use advanced search operators like `author:john` or `modified:since-1-week` for precise results.</Callout>
  </Tab>
</Tabs>
